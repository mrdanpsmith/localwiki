Ajax permissions error

You are seeing this error because your browser (most likely Google Chrome) is denying AJAX requests (or because you're viewing the source). If you are using Chrome launch it with the "--allow-files-to-access-files" command line option, use a different browser, or serve localwiki from a webserver.

Error loading the error page

You are seeing this error because there was an error loading the configured error page (or because you're viewing the source). That may sound confusing, but the next step is to make sure the error page that is configured in your config.js is accessible.
